# BBCode - Python IDE

基于 PyQt6 的现代化 Python 集成开发环境，专为 Python 学习和开发设计。

## ✨ 功能特性

### 📝 代码编辑器
- ✅ 行号显示
- ✅ Python 语法高亮（关键字、字符串、注释、函数、数字等）
- ✅ 实时语法检查（红色下划线标记错误行）
- ✅ 多标签页编辑
- ✅ 中文右键菜单（撤销、重做、剪切、复制、粘贴、删除、全选）
- ✅ 支持多种文件类型（.py, .txt, .json, .csv, .bat, .ps1, .md 等）

### 📁 文件浏览器
- ✅ 树形文件浏览
- ✅ 上一级文件夹导航
- ✅ 双击打开文件
- ✅ 路径显示
- ✅ 工具栏图标

### 🤖 AI 助手
- ✅ Ollama 本地模型支持
- ✅ **上下文记忆** - 支持连续对话，记住之前的对话内容
- ✅ **对话管理** - 清空对话、保存/加载对话历史
- ✅ **代码一键复制** - 代码块右上角显示复制按钮
- ✅ 流式实时响应
- ✅ 消息气泡界面
- ✅ 代码块高亮
- ✅ 强制中文回复
- ✅ 多行输入（Ctrl+Enter 发送）

### 💻 终端
- ✅ PowerShell 终端（支持系统命令）
- ✅ 运行代码时使用内置 Python 解释器
- ✅ demo 文件夹存放临时文件
- ✅ 程序关闭时自动清空 demo 文件夹
- ✅ 命令历史（上下键导航）
- ✅ 进程重启功能

### ⚙️ 设置
- ✅ 设置对话框（Ctrl+,）
- ✅ 通用设置（默认文件夹、自动保存、启动动画）
- ✅ AI 设置（Ollama 地址、默认模型、回复语言）
- ✅ 编辑器设置（字体大小、Tab 宽度、行号、语法检查）

### 🎨 界面
- ✅ 深色主题
- ✅ 启动动画
- ✅ 工具栏图标
- ✅ 可拖拽布局
- ✅ 自动保存布局状态

## 📂 项目结构

```
BBCode/
├── bbcode/                 # 核心代码包
│   ├── __init__.py        # 包初始化
│   ├── editor.py          # 代码编辑器（行号、语法高亮、语法检查）
│   ├── filebrowser.py     # 文件浏览器
│   ├── terminal.py        # 终端组件（PowerShell）
│   ├── ai_chat.py         # AI 聊天组件（上下文记忆、对话管理、代码复制）
│   ├── splash_screen.py   # 启动动画
│   ├── settings_dialog.py # 设置对话框
│   ├── logger.py          # 日志系统
│   └── main_window.py     # 主窗口
├── demo/                  # 临时文件存放目录（自动创建）
├── launcher.py            # 程序入口
├── BBCode.bat             # 启动脚本（控制台）
├── BBCodew.bat            # 启动脚本（无窗口）
├── installer.nsi          # NSIS 安装脚本
└── README.md              # 说明文档
```

## 🚀 快速开始

### 运行方式

#### 方式 1: 使用 launcher
```bash
python launcher.py
```

#### 方式 2: 使用批处理
```bash
BBCode.bat        # 带控制台窗口
BBCodew.bat       # 无控制台窗口
```

#### 方式 3: 使用内置 Python
```bash
python\python.exe launcher.py
```

## 🤖 AI 助手使用指南

### 1. 安装并启动 Ollama
```bash
ollama serve
```

### 2. 下载推荐模型
```bash
ollama pull qwen2.5-coder:7b
```

### 3. 使用 AI 助手
- 启动 BBCode，AI 助手会自动连接 Ollama
- 支持连续对话，AI 会记住之前的对话内容
- 点击「对话管理」可以清空、保存或加载对话历史
- 代码块右上角有「复制」按钮，一键复制代码

## 📦 依赖

- Python 3.8+
- PyQt6

安装依赖:
```bash
pip install PyQt6
```

## 📥 安装程序

### 下载安装
运行 `BBCode_Setup.exe` 进行安装，无需管理员权限。

### 从源码打包
使用 NSIS 创建安装程序:
```bash
"D:\app\windows\NSIS\makensis.exe" installer.nsi
```

## 📝 使用说明

### 运行 Python 代码
1. 在编辑器中编写代码
2. 按 F5 或点击工具栏运行按钮
3. 代码会在终端中执行，输出结果显示在终端面板

### 使用 AI 助手
1. 确保 Ollama 已启动
2. 在右侧 AI 助手面板输入问题
3. 按 Ctrl+Enter 发送消息
4. AI 会流式回复，支持代码高亮和一键复制

### 文件管理
- 左侧文件浏览器可以浏览文件夹
- 双击文件在编辑器中打开
- 支持多种文件类型的语法高亮
