# 云知识库功能使用说明

## 功能概述

云知识库功能允许您通过云端服务器同步和管理知识库内容。支持 Markdown 格式编辑、PDF/Word 文档导入转换、实时预览等功能。这使得团队可以共享知识库，并确保所有成员都能访问最新的知识内容。

## 主要特性

- ☁️ **云端同步**: 从云端服务器自动同步知识库内容
- 🔓 **无需认证**: 简化版服务器无需 API Key，开箱即用
- 📝 **Markdown 支持**: 完整的 Markdown 编辑、预览、导入导出
- 👁️ **实时预览**: 编辑时实时查看 Markdown 渲染效果
- 📄 **文档转换**: 支持 PDF、Word、TXT 导入并自动转换为 Markdown
- 🔄 **自动更新**: 支持手动和自动同步
- 📚 **AI 集成**: 同步的知识库会自动用于 AI 聊天上下文
- ⚙️ **灵活配置**: 可自定义服务器地址和同步选项
- 🌐 **局域网共享**: 支持多台设备通过局域网共享知识库


## 快速开始

### 1. 启动服务器

**安装依赖：**
```bash
cd cloud_kb_server
pip install -r requirements.txt
```

**启动服务器：**
```bash
# Windows
start_server.bat

# Linux/macOS
./start_server.sh

# 或直接使用 Python
python server.py
```

服务器启动后会显示访问地址：
```
🌐 访问地址:
   本机访问: http://127.0.0.1:5000
   本机访问: http://localhost:5000
   局域网访问: http://192.168.1.100:5000  ← 其他设备使用这个
```

### 2. 访问管理界面

打开浏览器访问 http://localhost:5000

管理界面功能：
- **Markdown 编辑器**：分屏编辑，左侧编辑右侧预览
- **工具栏**：快速插入 Markdown 语法（粗体、标题、列表、代码块等）
- **模板插入**：一键插入文章模板
- **文档导入**：支持 PDF、Word、TXT 导入转换
- **导入导出**：支持 JSON 和 Markdown 文件导入导出

### 3. 在 BBCode 中配置

1. 打开 BBCode 应用
2. 点击 **对话管理 ▼** → **知识库** → **云知识库设置...**
3. 输入服务器地址：`http://localhost:5000`
4. **API 密钥留空**（简化版无需认证）
5. 点击 **测试连接**，应该显示"连接成功（无需认证）"
6. 勾选 **启用云端同步**
7. 点击 **保存设置**
8. 点击 **立即同步** 开始同步

## 详细功能说明

### Markdown 编辑器

管理界面提供强大的 Markdown 编辑功能：

- **分屏编辑**：左侧编辑，右侧实时预览
- **工具栏**：快速插入常用 Markdown 语法
  - 粗体 (B)、斜体 (I)
  - 标题（H1-H3）
  - 列表（有序/无序）
  - 代码块 `{ }`
  - 引用、链接、图片
  - 分隔线
- **模板插入**：一键插入文章模板
- **GitHub 风格**：使用 GitHub Markdown CSS 样式

### 文档导入转换

支持将各种文档格式自动转换为 Markdown：

**支持的格式：**
- 📕 **PDF** (.pdf) - 提取文本并保留页面结构
- 📘 **Word** (.docx, .doc) - 保留标题、列表、表格格式
- 📄 **文本** (.txt) - 直接转换为 Markdown

**使用方法：**
1. 在管理界面点击"导入文档"按钮
2. 选择要导入的文件（PDF、Word 或 TXT）
3. 系统自动转换并创建知识条目
4. 转换后的内容保留原始文档的结构和格式

**转换特点：**
- 自动检测文档标题
- 保留段落和列表格式
- Word 表格转换为 Markdown 表格
- PDF 按页组织内容
- 添加导入来源标签

### 局域网共享

服务器支持局域网内多台设备共享使用：

**设置步骤：**
1. 确保服务器电脑和其他设备在同一 WiFi/网络
2. 在 BBCode 设置中使用局域网 IP（如 `http://192.168.1.100:5000`）
3. 其他设备即可连接并同步知识库

**注意事项：**
- 检查 Windows 防火墙，允许 Python 通过
- 服务器电脑需要保持运行
- 支持多人同时查看和同步

### 导入导出功能

**导出功能：**
- 点击"导出全部"按钮
- 下载 JSON 格式的备份文件
- 包含所有知识条目和元数据

**导入功能：**
- 支持导入 JSON 文件（从本系统导出）
- 支持导入 Markdown 文件（.md）
- 自动解析并创建知识条目

## 在 AI 聊天中使用

同步后的知识库会自动用于 AI 聊天：
- 当您发送消息时，系统会自动搜索相关知识
- 相关知识会被添加到 AI 的系统提示词中
- AI 会基于知识库内容提供更准确的回答

## 服务器配置选项

### 命令行参数

```bash
python server.py --host 0.0.0.0 --port 8080 --no-debug
```

- `--host`: 服务器主机地址（默认: 0.0.0.0）
- `--port`: 服务器端口（默认: 5000）
- `--no-debug`: 关闭调试模式

### 生产环境部署

**使用 Gunicorn（Linux/macOS）：**
```bash
pip install gunicorn
gunicorn -w 4 -b 0.0.0.0:5000 server:app
```

**使用 Waitress（Windows）：**
```bash
pip install waitress
waitress-serve --port=5000 server:app
```

## API 接口

云知识库客户端使用以下 REST API：

| 端点 | 方法 | 描述 |
|------|------|------|
| `/api/v1/health` | GET | 健康检查 |
| `/api/v1/info` | GET | 服务器信息 |
| `/api/v1/knowledge` | GET | 获取知识条目列表 |
| `/api/v1/knowledge` | POST | 创建知识条目 |
| `/api/v1/knowledge/{id}` | GET | 获取单个知识条目 |
| `/api/v1/knowledge/{id}` | PUT | 更新知识条目 |
| `/api/v1/knowledge/{id}` | DELETE | 删除知识条目 |
| `/api/v1/import` | POST | 导入文档（PDF/Word/TXT） |
| `/api/v1/categories` | GET | 获取分类列表 |
| `/api/v1/tags` | GET | 获取标签列表 |

## 故障排除

### 连接失败

**问题**: 测试连接时显示"连接失败"

**解决方案**:
1. 检查服务器地址是否正确
2. 确认服务器是否运行
3. 检查网络连接
4. 验证防火墙设置（允许 5000 端口）

### 同步失败

**问题**: 同步时显示"同步失败"

**解决方案**:
1. 检查服务器返回的错误信息
2. 确认本地知识库目录有写入权限
3. 检查服务器 API 是否正常工作
4. 查看服务器日志

### 知识库未生效

**问题**: AI 聊天中没有使用知识库内容

**解决方案**:
1. 确认知识库同步成功
2. 检查知识库文件是否正确下载到本地
3. 重新加载知识库（重启应用）
4. 检查查询内容是否与知识库内容相关

### 文档导入失败

**问题**: PDF/Word 导入失败

**解决方案**:
1. 确认已安装依赖：`pip install pymupdf python-docx`
2. 检查文件是否损坏
3. 尝试其他格式或文件
4. 查看服务器日志获取详细错误

## 配置存储

云知识库配置存储在 BBCode 的配置文件中：

- `cloud_kb.server_url`: 服务器地址
- `cloud_kb.api_key`: API 密钥（简化版留空）
- `cloud_kb.sync_enabled`: 是否启用同步

## 安全注意事项

⚠️ **注意**：简化版服务器没有认证机制，仅建议在以下场景使用：

- 本地开发环境
- 受信任的内网环境
- 个人使用

如需对外提供服务，建议：
1. 使用反向代理（Nginx/Apache）
2. 配置 HTTPS
3. 添加 IP 白名单
4. 或切换到需要认证的版本
