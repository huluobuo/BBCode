# Python 基础知识

## 变量和数据类型

Python 中的基本数据类型包括：
- int: 整数
- float: 浮点数
- str: 字符串
- bool: 布尔值
- list: 列表
- dict: 字典

## 条件语句

```python
if condition:
    # 代码块
elif another_condition:
    # 代码块
else:
    # 代码块
```

## 循环

for 循环示例：
```python
for item in iterable:
    # 处理每个元素
```

while 循环示例：
```python
while condition:
    # 循环体
```
