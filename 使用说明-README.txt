BBCode Portable - 使用说明
========================

这是 BBCode 的便携版本，无需安装 Python 即可运行。

启动方式：
---------
1. BBCode.bat     - 带控制台窗口启动（推荐，可以看到调试输出）
2. BBCodew.bat    - 不带控制台窗口启动

注意事项：
---------
- 请使用 BBCode.bat 或 BBCodew.bat 启动程序
- 不要直接运行 Scripts 目录下的 thonny.exe
- 首次启动可能需要几秒钟加载时间

文件说明：
---------
- python/         - Python 解释器和必要工具
- Lib/            - Python 库文件
- thonny/         - BBCode/Thonny 源代码
- plugins/        - 插件目录
- res/            - 资源文件

如果遇到问题：
-------------
1. 确保解压路径不包含中文或特殊字符
2. 确保有写入权限（用于保存配置）
3. 尝试以管理员身份运行

========================
BBCode Portable Usage
========================

This is a portable version of BBCode that runs without Python installation.

Launch Methods:
--------------
1. BBCode.bat     - Launch with console window (recommended for debugging)
2. BBCodew.bat    - Launch without console window

Notes:
------
- Please use BBCode.bat or BBCodew.bat to start the program
- Do not run python.exe directly from the python folder
- First launch may take a few seconds to load

Troubleshooting:
---------------
1. Ensure extraction path doesn't contain Chinese or special characters
2. Ensure you have write permissions (for saving configurations)
3. Try running as administrator if issues occur
