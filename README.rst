BBCode - 智能AI辅助的Python IDE
================================

BBCode是一个基于Thonny的Python IDE，它添加了一些AI功能，如智能代码补全、智能AI辅助代码等。
此IDE集成了python3.11.9+node.js环境，支持在纯净的Windows环境下运行。

当前项目版本为1.0.1.11

使用说明：直接打开BBCode.bat或BBCodew.bat

特性亮点
--------

🤖 **AI智能功能**
- 智能代码补全 - AI驱动的上下文感知代码建议
- AI辅助代码生成 - 基于自然语言描述的代码生成
- 智能错误修复 - AI自动检测并建议修复代码错误
- 代码解释功能 - AI解释复杂代码逻辑

⚡ **集成开发环境**
- 基于Thonny的稳定Python IDE基础
- 内置Python 3.11.9解释器
- 集成Node.js运行环境
- 完整的调试工具链
- 语法高亮和代码折叠

🎨 **用户体验**
- 深色/浅色主题切换
- 中文界面友好支持
- 简洁直观的操作界面
- 教育用途优化设计

🔧 **开发工具**
- 内置包管理器（pip）
- 变量查看器
- 代码分析工具
- 项目管理功能

系统要求
--------

- **操作系统**: Windows 7/8/10/11 (64位)
- **内存**: 最低4GB RAM，推荐8GB或以上
- **磁盘空间**: 至少500MB可用空间
- **网络连接**: 需要网络连接以使用AI功能

快速开始
--------

### 下载安装

1. 从[发布页面](https://github.com/huluobuo/BBCode/releases)下载最新压缩包
2. 解压到任意目录（建议不要放在系统盘）
3. 双击运行 `BBCode.bat` (带控制台) 或 `BBCodew.bat` (无控制台)

### 首次使用

1. 启动BBCode后，系统会自动初始化Python和Node.js环境
2. 创建新Python文件或打开现有项目
3. 开始享受AI辅助编程体验！

AI功能使用指南
--------------

### 智能代码补全

在编写代码时，BBCode会自动提供AI驱动的代码建议：

- 输入函数名时显示参数提示
- 根据上下文推荐相关代码片段
- 自动导入所需的模块

### AI代码生成

使用自然语言描述您想要的功能：

```
# 输入：创建一个函数，计算两个数的和并返回结果
# AI会自动生成：
def add_numbers(a, b):
    """计算两个数的和"""
    return a + b
```

### 错误检测与修复

当代码出现错误时，AI会：

- 自动检测语法和逻辑错误
- 提供详细的错误解释
- 建议修复方案
- 一键应用修复

项目结构
--------

```
BBCode/
├── thonny/           # Thonny核心框架
├── plugins/          # 插件系统
│   ├── bbcode_info.py    # BBCode信息面板
│   ├── chat.py           # AI聊天功能
│   └── ai_assistant.py   # AI助手插件
├── python/           # 内置Python 3.11.9环境
├── nodejs/          # Node.js运行环境
├── res/             # 资源文件
│   ├── icons/       # 图标资源
│   └── themes/      # 主题文件
├── BBCode.bat       # 主启动脚本（控制台）
├── BBCodew.bat      # 启动脚本（无控制台）
├── launcher.py      # 程序启动器
└── VERSION          # 版本信息
```

常见问题
--------

### Q: AI功能需要网络连接吗？
A: 是的，AI功能需要网络连接来访问云端AI服务。

### Q: 支持哪些Python版本？
A: 内置Python 3.11.9，同时兼容Python 3.9-3.13项目。

### Q: 如何更新AI模型？
A: 系统会自动更新AI模型，无需手动操作。

### Q: 是否支持自定义AI服务？
A: 当前版本使用预设的AI服务，未来版本将支持自定义配置。

### Q: 如何处理隐私和数据安全？
A: 代码数据仅在本地处理，AI服务仅接收必要的代码片段进行分析。

故障排除
--------

### 启动问题

1. **无法启动**: 检查系统是否满足最低要求，尝试以管理员身份运行
2. **缺少依赖**: 确保解压完整，不要删除任何文件
3. **权限问题**: 将BBCode安装到非系统目录

### AI功能问题

1. **AI无响应**: 检查网络连接，重启程序
2. **代码生成慢**: 可能是网络延迟，稍等片刻
3. **功能异常**: 查看日志文件或联系技术支持
